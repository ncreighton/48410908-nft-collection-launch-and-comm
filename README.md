# NFT Collection Launch and Community Management Discord Bot

A Discord bot for NFT project teams and Web3 communities managing collection launches and holder engagement. Automates whitelist verification via wallet signature authentication, posts countdown timers for mint events, verifies NFT ownership for role gating, runs holder-exclusive raffles and giveaways, tracks floor price and volume from OpenSea and Blur APIs with hourly alerts, and generates weekly community analytics reports on member growth and engagement rates.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
